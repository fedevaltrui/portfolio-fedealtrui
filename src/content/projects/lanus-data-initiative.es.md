# Museo del Club Atlético Lanús

**Descripción general:**
Arquitectura integral de portal para manejar vastos volúmenes de estadísticas de jugadores y partidos, escalando APIs de alto rendimiento para uso público e interno.

Este proyecto implicó diseñar esquemas robustos y construir arquitecturas de frontend y backend para facilitar el acceso a datos y la creación de informes para el Club Atlético Lanús.

## Funcionalidades a destacar más adelante

* Estadísticas detalladas de jugadores
* Datos de partidos
* Métricas de rendimiento
* ...
